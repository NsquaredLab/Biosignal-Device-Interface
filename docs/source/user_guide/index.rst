Overview
========

The Biosignal Device Interface provides a unified Python API for communicating with various biosignal acquisition devices. It's designed with the following principles:

**Ease of Use**
    Simple, intuitive API that gets you connected to devices quickly.

**Real-time Performance**
    Optimized for low-latency data streaming and real-time applications.

**GUI Integration**
    Ready-to-use PySide6 widgets for rapid application development.

**Extensibility**
    Modular architecture that makes it easy to add support for new devices.

**Cross-platform**
    Works seamlessly on Windows, macOS, and Linux.
